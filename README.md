Every agent follows:

- **No `any`** â€” Types are documentation
- **No `throw`** â€” Use `Result<T, E>`
- **No god classes** â€” Single responsibility
- **No code without tests** â€” TDD mindset
- **No implementation without spec** â€” Contracts first
